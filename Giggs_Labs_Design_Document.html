01. System Architecture Overview

The Giggs Software Labs website is a full-stack application with a Next.js frontend deployed on Vercel and a Django REST backend deployed on Render. The frontend uses server-side rendering (SSR) to fetch dynamic content from the backend API.

10. Technical Decisions

DecisionRationale
Next.js Pages RouterFile-based routing with SSR support for SEO and dynamic content
Django REST FrameworkRapid API development with built-in admin, serializers, and auth
Framer MotionProduction-grade animations with React — scroll-reveal, page transitions
Tailwind CSSUtility-first styling for rapid iteration with custom design tokens
Lucide ReactConsistent, lightweight, tree-shakeable icon system
PageShell patternDRY layout component reused across all pages for consistency
Client-side chatbotZero API dependency — keyword matching runs entirely in-browser
Video hero backgroundsPremium feel with per-tab crossfade for each service pillar
